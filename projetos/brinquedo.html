Projeto: Catálogo de Brinquedos

Este projeto foi desenvolvido com o objetivo de apresentar uma plataforma colorida e interativa para exibir diferentes tipos de brinquedos. O design transmite diversão e leveza, com foco na navegação simples e intuitiva.

Tecnologias Utilizadas

  • HTML5
  • CSS3
  • JavaScript
  • Spring Boot
  • Java
  • PHP
  • Figma

Minha Participação

Fui responsável pela prototipagem no Figma, criação do layout principal e organização dos cards dos brinquedos.
