Sobre Mim

FrontEnd

Prazer, me chamo Gustavo, mas se preferir pode me chamar de Gu. Tenho 20 anos e sou um amante da tecnologia. Atualmente estou cursando Desenvolvimento de Software e Multiplataformas. Estou no 5º semestre e em busca de novos desafios, como o primeiro trabalho na área, especialmente na área de Front-End. Projetos do semestre



Petlife

PetLife é um projeto desenvolvido para facilitar o cuidado e o bem-estar dos pets. Com uma interface amigável e intuitiva, nosso site oferece uma plataforma completa para donos de animais de estimação.

Pokemon API

Pokémon API Explorer é um projeto desenvolvido para fornecer informações detalhadas sobre o universo Pokémon, utilizando uma API robusta e bem estruturada. Ideal para desenvolvedores e entusiastas de Pokémon que desejam explorar dados de forma rápida e eficiente.

Barbearia

O projeto do site da barbearia foi desenvolvido com o objetivo de oferecer praticidade e modernidade aos clientes. Através dele, é possível conhecer os serviços disponíveis, agendar horários de forma rápida e segura, além de visualizar informações sobre a equipe e promoções. O design foi pensado para transmitir elegância e confiança, utilizando cores que refletem a identidade da barbearia e garantindo uma navegação simples e intuitiva.

Catalogo brinquedo

O projeto do site de catálogo de brinquedos foi desenvolvido com o objetivo de oferecer uma plataforma prática e interativa para apresentar diferentes tipos de brinquedos. A ideia é facilitar a navegação dos usuários, permitindo que encontrem facilmente opções por categoria, idade ou estilo. O design foi pensado para ser colorido e atrativo, transmitindo diversão e leveza, ao mesmo tempo em que garante uma experiência simples e intuitiva.

Loja completa PetShop

O projeto do aplicativo mobile de pet shop foi criado para facilitar a vida dos clientes que buscam serviços e produtos para seus animais de estimação. Com uma interface intuitiva e moderna, o app permite agendar banhos, tosa e consultas veterinárias, além de acompanhar pedidos e promoções exclusivas. O design prioriza a praticidade no uso pelo celular, garantindo uma experiência rápida, organizada e acessível em qualquer lugar.
